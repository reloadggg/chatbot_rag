## 🚀 新功能详解

### 📁 文件上传和处理

#### 支持的文件格式
- **PDF文件**: 自动文本提取，支持多页文档
- **文本文件**: TXT、MD格式，支持多种编码
- **JSON文件**: 结构化数据转换为可读文本
- **文件大小**: 最大10MB，确保系统性能

#### 智能文档处理流程
1. **文件上传** → 验证格式和大小
2. **文本提取** → 根据文件类型提取内容
3. **智能分块** → 1000字符块，200字符重叠
4. **向量化存储** → 自动添加到向量数据库
5. **知识库构建** → 立即可用于问答

## 🌐 新增API端点

### 文件管理API
```bash
# 上传文件到知识库
POST /upload
- 支持多格式: PDF, TXT, MD, JSON
- 最大大小: 10MB
- 自动处理: 提取 → 分块 → 向量化

# 获取文档列表
GET /documents
- 返回所有已上传文档信息

# 删除指定文档
DELETE /documents/{file_id}
- 删除文档文件

# 获取统计信息
GET /documents/stats
- 文档数量、总大小、向量统计
```
